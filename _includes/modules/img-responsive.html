Examples

Mobile first

Start by loading the smallest image and only load larger if needed. If the data-src attr is not set then no image replacement happens.

<img src='/image/phone.jpg' data-src='{ xs:"/image/phone.jpg", sm:"/image/tablet.jpg", md:"/image/laptop.jpg", lg:"/image/desktop.jpg"}' />

$('img').ImgResponsive();
