Has Icon/Button

This component allows for easy adding of buttons and icons within an any element. Place any icon using the <i> tag or any button with class .btn inside another element and add either has-icon or has-btn classes. By default the icons and buttons will be left aligned, to make them right aligned use has-right.

To inset the icon or button from the left or right use the attribute data-offset-pos="10". To add the inset padding to the parent element use the attribute data-pad-parent="true".

<div>
  <div class='has-icon' data-pad-parent='1'>
    <i class='fa fa-plus-square'></i> <h2>Title with Icon - Padding parent</h2>
  </div>
  <p>This text will now be offset as well!</p>
</div>

<div class="alert alert-success has-btn has-right">
	<a href="#" class="btn btn-success btn-lg">Click Me</a> ..and so are button sizes!
</div>
