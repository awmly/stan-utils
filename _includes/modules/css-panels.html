Panels

Panels are a great way to contain components such as collpases, tabs or navigations but also for standard text and image e.g latest blog post. To create a panel use the class sa-panel and to then make it a block panel add the class sa-block.

Panels are based around the asumption that there will be a title and content and these should be enclosed within the classes sa-title and sa-content. Within a title use the tag h6 to use the predefined styling.

Panels come with pre-defined spacing/size presets to apply one of these add the class sa-xs, sa-sm, sa-md, sa-lg or sa-xl.

<div class="sa-panel">
  <div class="sa-title">
    <h6>Title</h6>
  </div>
  <div class="sa-content">
    Content
  </div>
</div>
