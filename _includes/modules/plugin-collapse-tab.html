Collapse Tab

Combines Bootstraps tab and collapse widgets to be responsive; shows tabs on larger screen devices and collapse on smaller screens.

  • Examples
  • Settings
  • Methods
  • Events

Examples

Tabs for tablets and desktop - Collapse for mobile

The tabs/collapses on this page are also a working example.

<div class="collapse-tab">
	<ul class="nav nav-tabs hidden-xs">
		<li class="active"><a href="#ct1" data-toggle="tab">CT1</a></li>
		<li><a href="#ct2" data-toggle="tab">CT2</a></li>
	</ul>
	<div class="tab-content">
		<div class="tab-pane active" id="ct1">
			<h3 data-toggle="collapse" data-target="#ct1>div">CT1</h3>
			<div class="collapse">...</div>
		</div>
		<div class="tab-pane active" id="ct2">
			<h3 data-toggle="collapse" data-target="#ct2>div">CT2</h3>
			<div class="collapse">...</div>
		</div>
	</div>
</div>

<script>
$('.collapse-tab').CollapseTab({
	breakpoint:768
});
</script>

View Example

Settings

NameTypeDefaultDescription
breakpoint integer 768 The breakpoint for when tabs and collapse switch

Methods

$('#selector').CollapseTab('resize'); //Invokes the resize method

Events

$('#selector').on('tab.sa.collapsetab',function(event,settings){
	// Triggered every time view switches to tabs
});

$('#selector').on('collapse.sa.collapsetab',function(event,settings){
	// Triggered every time view switches to collapse
});

Standard Bootsrap tab and collapse events are also fired.
