Media Query Class

Sets class or attribute values depending on the current window size. This is a core component of STAN plugins as it is used by other modules to determine the current device type.

Examples

Bootstrap body classes

Add classes and data values to body based on Bootstraps media query break points.

$('body').MQClass({
{{ site.tab }}breakpoints:[
{{ site.tab }}{{ site.tab }}{ 'min-width':0, 'max-width':768, 'class':'device-xs', 'data-current-device':'xs' },
{{ site.tab }}{{ site.tab }}{ 'min-width':768, 'max-width':992, 'class':'device-sm', 'data-current-device':'sm' },
{{ site.tab }}{{ site.tab }}{ 'min-width':992, 'max-width':1200, 'class':'device-md', 'data-current-device':'md' },
{{ site.tab }}{{ site.tab }}{ 'min-width':1200, 'max-width':0, 'class':'device-lg', 'data-current-device':'lg' }
{{ site.tab }}]
});

Settings

NameTypeDefaultDescription
breakpoints array false an array of breakpoints as objects
breakpoints.min-width integer 0 set the min width of breakpoint
breakpoints.max-width integer 0 set the max width of breakpoint
breakpoints.class string '' set the class to add
breakpoints.data-* string '' set attributes to add

Methods

$('#selector').MQClass('resize'); Invokes the resize method
