Alerts

Taking Bootstrap alerts as its base this component allows for easy adding of buttons and icons within an alert box. Place any icon using the <i> tag or any button with class .btn inside an alert and add either has-icon or has-btn classes to the alert. By default the icons and buttons will be left aligned, to make them right aligned use icon-right or btn-right.

<div class="alert alert-success has-icon icon-right">
	<i class="fa fa-check fa-4x"></i> All icon sizes are handled...
</div>

<div class="alert alert-success has-btn">
	<a href="#" class="btn btn-success btn-lg">Click Me</a> ..and so are button sizes!
</div>
