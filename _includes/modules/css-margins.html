Margin/Paddings

Add or remove margins and paddings for requited elements.

To remove the padding or margins from all sides use .no-pad or .no-mar to only select certain sides add either -top, -btm, -left or -right to the class name.

To add a 30px padding or margin use .mar or .pad to only select certain sides add either -top, -btm, -left or -right to the class name. Additionaly use a -lrg suffix to increase the value to 60px or -sml to decrease the value to 15px.

<div class="mar-top"></div>
<div class="mar-top-sml"></div>
<div class="mar-lrg"></div>
<div class="pad"></div>
<div class="no-pad-left"></div>
