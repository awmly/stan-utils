Checkbox

A checkbox module that allows any element to be used as the checkbox. To have a checkbox checked when the page loads simply set the value of the hidden input to 1.

<div class="sa-checkbox">
  <i class='fa fa-square-o sa-off sa-inset pad-right-xs'></i>
  <i class='fa fa-check-square-o sa-on sa-inset pad-right-xs'></i>
  <label>An icon based checkbox</label>
  <input type='hidden' name='cbox' value='' />
</div>
