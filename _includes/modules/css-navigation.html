Navigation

Provides base styles for 4 main types of navigation. Add either nav-horizontal, nav-horizontal-block, nav-vertical or nav-vertical-block to any ul to create a navigation. Icons can be added to either of the horizontal navs without any extra classes but to use icons in a vertical nav add the class nav-vertical-icon.

Sub navigation can be added to either of the the block navs, just start another ul inside an li. Apply the class fix-width to any sub nav ul to enable the sub nav to be its natural width.

<ul class="nav-horizontal-block">
  <h2>Nav Title</h2>
  <li><a href=''>Link 1</a></li>
  <li><a href=''>Link 2</a>
    <ul class="fix-width">
      <li><a href=''>Sub Nav Link 1</a></li>
      <li><a href=''>Sub Nav Link 2</a></li>
    </ul>
  </li>
</ul>

<ul class="nav-vertical nav-vertical-icon">
  <li><a href=''><i class="fa fa-check"></i> Link 1</a></li>
  <li><a href=''><i class="fa fa-check"></i> Link 2</a></li>
</ul>
