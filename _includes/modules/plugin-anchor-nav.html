Examples

Sub Title Nav

Creates a navigation which anchor links to all Sub Titles found on a page. This plugin works best with the ScrollTo plugin, simply chain both plugins on the required element.

<ul class="sidebar">
</ul>

<script>
$('.sidebar').AnchorNav({
	items:'h2',
});	
</script>

Settings

NameTypeDefaultDescription

a_class string Add a class to the automatically added <a>'s
add_li_active_class boolean false Add an active class to the <li> currently in view.
add_a_active_class boolean true Add an active class to the <a> currently in view.
