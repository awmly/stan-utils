Collapse

Built upon BS collapse this component aids in changing the display depending on if the collapse is open or closed. Any element within the main sa-collapse element with the class sa-off will be displayed when the collapse is closed and any element with the class sa-on will be displayed when the collapse is open. An active class is also applied to sa-collapse when the collapse is open, you can also make a collapse start open by loading the page with the active class already applied.

Any element with the class sa-click can be used to activate the collapse mechanism and any element with the class sa-content will behave as a collapse.

<div class="sa-collapse">

  <div class="sa-content">
    ...
  <div>

  <p class="sa-click sa-on"><a href="#">Read less...</a></p>
  <p class="sa-click sa-off"><a href="#">Read more...</a></p>

</div>
