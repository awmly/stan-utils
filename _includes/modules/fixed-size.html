Examples

Responsive Columns

Fix the heights of columns when inside a responsive column structure.

$('.row').FixedSize({
{{ site.tab }}selector:'.col>div',
{{ site.tab }}devices:{ xs:false, sm:2, md:3, lg:4 }
});

View Example

Settings

NameTypeDefaultDescription
selector string div The selectors to fix the size of
devices object { xs:0, sm:0, md:0, lg:0 } How many selectors to group together when fixing the size. Set to 0 for all, or set to a number. ie setting to 3 will make the first 3 selectors the same height, then the next 3 and so on. Devices based on Bootstrap spec.

Methods

$('#selector').FixedSize('fix'); Invokes the fix method

Events

$('#selector').on('sa.fixedsize.fix',function(event,settings){
{{site.tab}} //Triggered every time selectors are resized
});
