Image Box

  • fill-full Image will always fill the container meaning not all the image will be seen at all times. Enlarging of image will occor if container is larger than image, the height and width of the image will be the same or greater than the container at all times.
  • fill-max Image will always fill the container meaning not all the image will be seen at all times. If image is smaller than container it will be positioned centrally.
  • fit-full Image will always fit inside the container meaning all the image will be seen at all times. Enlarging of image will occor if container is larger than image, either the height or width of the container and image will be the same at all times.
  • fit-max Image will always fit inside the container meaning all the image will be seen at all times. If image is smaller than container it will be positioned centrally.
